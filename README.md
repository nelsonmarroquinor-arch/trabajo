# Taller Semana 2 - Ciclo For en Java

## Descripción General
Este proyecto contiene la solución a 5 ejercicios prácticos sobre el uso del ciclo `for` en Java, incluyendo:
- For simple para recorridos y acumuladores
- For anidado para patrones y matrices
- Razonamiento lógico con estructuras bidimensionales

## Requisitos
- Java 17
- Maven

## Estructura del Proyecto

## Cómo Ejecutar
1. Compilar: `mvn clean compile`
2. Ejecutar: `mvn exec:java`

## Descripción de Ejercicios

### Ejercicio 1: For Simple
- Muestra números del 1 al 20
- Muestra números pares del 1 al 50
- Muestra conteo descendente del 20 al 1

### Ejercicio 2: Arreglo de 10 Enteros
- Genera arreglo con números aleatorios
- Calcula suma total
- Calcula promedio
- Encuentra el número mayor

### Ejercicio 3: Patrón Numérico
- Patrón triangular incremental
- Patrón con números repetidos
- Patrón de pirámide invertida

### Ejercicio 4: Matriz 3x3
- Llena matriz con números del 1 al 9
- Muestra en formato tabla
- Muestra diagonal principal

### Ejercicio 5: Matriz 4x4
- Matriz con números aleatorios
- Calcula suma por cada fila
- Calcula suma total de la matriz
- (Extra) Calcula suma por columnas

  
## Reflexión Personal

Este taller me permitió comprender a profundidad el funcionamiento del ciclo for en Java. Los ejercicios incrementaron gradualmente su complejidad, comenzando con recorridos simples hasta llegar a estructuras bidimensionales con lógica de acumulación.

El uso de for anidados para crear patrones numéricos resultó especialmente útil para visualizar cómo funcionan los ciclos internos y externos. Las matrices 3x3 y 4x4 me ayudaron a entender cómo acceder y manipular datos en dos dimensiones, así como a implementar algoritmos de suma por filas y columnas.

La práctica con acumuladores y cálculos estadísticos (suma, promedio, número mayor) reforzó conceptos fundamentales de programación. Además, la implementación de la matriz 4x4 me permitió desarrollar pensamiento lógico para resolver problemas que requieren recorrer estructuras de datos complejas.

Considero que estos ejercicios son una base sólida para abordar problemas más complejos que involucren iteraciones y manejo de datos en arreglos multidimensionales. La comprensión de estos conceptos es esencial para el desarrollo de aplicaciones más avanzadas.
